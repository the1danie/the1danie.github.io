Задание 1
Файл находится в Maps/map2.html. Svg файл Map_Of_Belarus_blank.svg. Беларусь Природная Заповедники

Проект включает в себя HTML-страницу с SVG-изображением карты, на которой отмечены различные части Беларусии, представленные в виде разных цветовых областей. При наведении курсора на каждую область появляется описание этой области. Описание загружается динамически с помощью JavaScript.

- `map2.html`: основной HTML-файл, содержащий структуру веб-страницы и интерактивную карту
- `Map_Of_Belarus_blank.svg`: изображение пустой карты Беларусии
- `style.css`: файл со стилями для веб-страницы
- `script.js`: файл со скриптом JavaScript для обработки событий наведения курсора и отображения описания областей
При наведении на зеленый участок, появляется информация о заповеднике, расположение, и Площадь земли

Задание 2
Файл находится в Maps/task2/cat.html

Этот проект представляет собой интерактивный рассказ о коте и мышах с использованием HTML, CSS и JavaScript.

Проект включает в себя веб-страницу, на которой рассказывается история о коте и мышах. В тексте рассказа определенные слова, относящиеся к коту, выделены классом `highlight`. При наведении курсора на эти слова, они заменяются на изображение кота.

- `cat.html`: основной HTML-файл, содержащий структуру веб-страницы и текст рассказа
- `style.css`: файл со стилями для веб-страницы
- `script.js`: файл со скриптом JavaScript для обработки событий наведения курсора и замены текста на изображение
- `cat.png`: изображение кота, используемое как фон на странице
- `cat and mouse.jpg`: изображение кота с мышью, которое заменяет текст при наведении курсора на слова, относящиеся к коту
Был использован метод JQuery

Файл находится в Diagram/task1.html.
Js code находится в Diagram/chart.js
Css находится в Diagram/style.css

Брал информацию об альткоинах в данный момент. Цена на данный момент, и объем криптовалюты.
Подключение шрифта Open Sans с помощью Google Fonts для стилизации текста.
Использование библиотеки Chart.js для создания графика цен.
Подключение внешнего файла стилей style.css для определения внешнего вида элементов на странице.
На странице размещены блоки с информацией о текущих ценах нескольких альткоинов (Ethereum, BNB, Litecoin, Solana, AAve, Gnosis, Montero), а также их суточном объеме торгов.
Создан график цен с использованием Chart.js. График представляет собой линейный график, отображающий цены альткоинов.
Эта HTML-страница с графиком цен и информацией о криптовалютах предоставляет пользователю удобный способ отслеживать текущие цены и суточный объем торгов различных криптовалют.
